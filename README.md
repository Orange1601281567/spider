# spider
## 项目简介
一个后台管理的后台框架模板，用于系统的敏捷开发


## 涉及内容
layui springboot spring security mybatis-plus 

## 访问地址
请自行查看yaml文件<br>
  
## 功能简介
这个系统中包括以下功能
<br>
1.用户管理<br>
2.菜单管理<br>
3.日志管理<br>
4.角色管理<br>
5.代码生成<br>
 
### 用户管理
管理员用户列表的增删改查

### 菜单管理
管理员可对菜单进行增删改查操作(由于技术有限,只能做成二级菜单)

### 日志管理
管理员查看用户的所有操作具体信息(通过切面aop的方式实现)

### 角色管理
管理员可以增删改查角色信息

 ### 代码生成
通过系统的测试类配置数据源信息以及需要自动生成的表名即可自动生成接口代码
